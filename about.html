Technologies Used

These are the three foundational technologies we used to create DeskSpacing.com

Bootstrap 5

Bootstrap was used to create the user interface of DeskSpacing.com, including the landing page.

ThreeJS

ThreeJS powers the 3D rendering on DeskSpacing.com, allowing us to manipulate 3D models directly in the browser.

Blender

Blender was used to create and design all of the 3D models used on DeskSpacing.com.
